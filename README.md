# Rasterizer
Software Rasterizer using CPU only. Applicaton uses
ImGui and OpenGL to show result in window;

## Getting Started
Visual Studio 2019 or 2022 is recommended.   
To setup projects run one of the scripts in <b>Scripts</b> folder.
